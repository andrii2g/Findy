# Findy

Findy is a simple mobile app for iOS and Android that helps you remember where household things are stored by saving a short description and a photo of the place where each item belongs.

## User main workflows

### Add a new thing

1. Open Findy.
2. Tap **Add new thing**.
3. Enter a short description of the item.
4. Take a photo of the place where the item is stored.
5. Save the record locally on the device.

### Find a thing

1. Open Findy.
2. Browse the saved list or use search.
3. Tap the item you want to find.
4. View the saved description and location photo to see where it belongs.
